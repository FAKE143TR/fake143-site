Code Example

Typical interaction pattern for a multi-chain operation:

// Initialize agent with chain configurations
const agent = await FAKEAI.init({
  chains: ["ethereum", "arbitrum", "solana"],
  mode: "autonomous"
});

// Execute cross-chain portfolio check
const portfolio = await agent.getPortfolio({
  include: ["tokens", "nfts", "defi-positions"],
  chains: "all"
});

// Monitor for opportunities
agent.on("opportunity", (opp) => {
  if (opp.risk < opp.threshold) {
    agent.execute(opp.action);
  }
});
